# Security Policy

## Supported Versions

Security fixes are applied on the `main` branch and current preview releases.

## Reporting a Vulnerability

Please report vulnerabilities privately:
- Open a private GitHub security advisory when possible
- Or email: security@kamibot.dev

Please include:
- Affected version/commit
- Reproduction steps
- Impact assessment
- Suggested mitigation (if known)

We aim to acknowledge reports within 72 hours.
